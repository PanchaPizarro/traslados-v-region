# Traslados V Región → Aeropuerto SCL

Sitio web de un servicio de **transfer privado** desde la V Región (Viña del Mar,
Valparaíso, Concón, Quilpué, Villa Alemana) hacia el Aeropuerto de Santiago (SCL).

Página estática de una sola sección (single page), responsive, con botón de
contacto directo por WhatsApp.

## Estructura

```
index.html              Página principal
assets/css/styles.css   Estilos
assets/img/exterior/    Fotos del vehículo (con y sin fondo)
assets/img/interior/    Fotos del interior
```

## Cómo verla

Abre `index.html` en cualquier navegador (no requiere servidor).

## Personalizar

- **Nombre del negocio:** busca `Traslados V Región` en `index.html`.
- **WhatsApp / teléfono:** busca `56949044873` en `index.html`.
- **Rutas y tarifas:** edita la sección `#rutas` en `index.html`.

## Publicar (GitHub Pages)

En el repositorio: *Settings → Pages → Branch: `main` / root → Save*.
La web queda en `https://panchapizarro.github.io/<nombre-repo>/`.
